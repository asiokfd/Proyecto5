## Proyecto de Machine Learning 

#### En este proyecto el objetivo es, dado un dataset, predecir el precio de los diamantes. Para esto se ha organizado una competición en [kaggle]( https://www.kaggle.com/c/diamonds-datamad0321) en la que iremos subiendo nuestras distintas predicciones.

#### La estructura es muy sencilla, primero he realizado una categorización de los datos, y después he probado modelos. Una vez elegido el mejor modelo he realizado pruebas con los hiperparametros. Todo el trabajo se encuentra en un jupyter notebook llamado "main", utilizando "support" como apoyo para algunas tareas.
